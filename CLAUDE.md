# CLAUDE.md

## Commands

| Purpose | Command |
|---------|---------|
| Build | `npm run build` |
| Unit tests | `npm run test:unit` |
| Type check | `npm run test:types` |
| Format check | `npm run format` |
| Format fix | `npm run format:fix` |

## Rules

- Unit tests (`npm run test:unit`) can be run without asking permission
- Use `npm run format:fix` to fix formatting, not `npx prettier --write`
